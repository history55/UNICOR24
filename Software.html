SOFTWARE 

El software se refiere a programas, instrucciones y datos que permiten a una computadora realizar tareas específicas. 

Se pueden distinguir sistemas operativos, como Windows, aplicaciones, programas de productividad, videojuegos y cualquier otro software de ejecución. 

El software de sistema o sistema de operación es un conjunto de programas y herramientas que manejan los recursos de hardware, y permiten que otros programas se ejecuten en una computadora. 

Ejemplos comunes de sistemas operativos incluyen:  Windows, macOS, Linux, Android e iOS. Por otro lado, se entiende por lenguaje de programación a un conjunto de reglas y símbolos utilizados para escribir un programa de computadora. 

Estos lenguajes permiten a los programadores comunicarse con las computadoras y enviarles instrucciones sobre cómo realizar varias tareas.

Los lenguajes de programación se caracterizan por una variedad de características, pero algunas de las más comunes incluyen la sintaxis, que es un conjunto de reglas que definen la estructura y la gramática del lenguaje, la semántica, que se refiere al significado de las instrucciones escritas en determinado lenguaje; el tipado, que es la especificación de cómo un lenguaje maneja los diversos tipos de datos, por ejemplo enteros, flotantes, cadenas, etc.; la orientación ya que algunos lenguajes son orientados a objetos, otros funcionales, etc.; la portabilidad, mecanismo que permite que una aplicación escrita en determinado lenguaje de manera tal que pueda ser ejecutada en un sistema distinto. 

Finalmente se puede mencionar la eficiencia que se asocia a la capacidad del lenguaje de desarrollar programas rápidamente, sin utilizar mucha memoria ni téras de procesador y las bibliotecas o frameworks, que son conjuntos de funciones preprogramadas que permiten realizar aplicaciones de manera más rápida.
